DOM Questions & Answers

1. What is the DOM?

The DOM (Document Object Model) is an interface that allows JavaScript to interact with HTML and XML documents. It represents the structure of the document as a tree of objects.

2. How can you access an element in the DOM?

You can access an element in the DOM using methods like getElementById(), querySelector(), or getElementsByClassName().

3. What is the difference between the DOM and BOM?

The DOM (Document Object Model) refers to the document structure (HTML or XML), while the BOM (Browser Object Model) allows access to browser features like the window, history, and navigation.

5. How can you add an element to the DOM?

You can add a new element to the DOM using methods like appendChild() or insertBefore(). You first create the element with document.createElement(), then append it to a parent.
